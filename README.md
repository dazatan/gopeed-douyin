**Gopeed 抖音视频下载扩展**

**声明**

本项目代码全部开源，解析代码使用的是网络公开代码，本插件所有代码都来自AI（deep seek），本插件仅供学习交流使用，不得用于商业用途，如有侵权请联系作者删除

**安装**

打开Gopeed扩展页面，输入 https://github.com/dazatan/gopeed-douyin  点击安装即可。

**使用说明**

安装完扩展后，在任务创建页面输入抖音视频链接，点击创建任务即可解析下载，效果图：
